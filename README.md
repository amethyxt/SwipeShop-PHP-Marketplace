# SwipeShop

## Project Overview
SwipeShop is a web application developed as a group project for academic purposes.
The project focuses on shopping-related features and improving user experience through a simple and easy-to-use web interface.

- API integration (basic)
- HTML, CSS, JavaScript / PHP (depending on project setup)
